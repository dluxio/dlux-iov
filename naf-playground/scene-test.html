How It Works

When you click on a test link, the application will:

  1. Load the specified scene file from the server
  2. Convert the JSON scene data to application state
  3. Update all entities in the 3D scene
  4. Update the Monaco code editor to show the scene HTML representation

This demonstrates the complete pipeline from an external scene definition to both visualization and code representation.

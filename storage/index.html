IPFS Pinning

SPK Network

Store your files on the InterPlanetary File System with the SPK Network Proof of Storage system. Simply Power Up BROCA to earn BROCA Resource Credits, which are used to host your files on the DePIN storage nodes. The BROCA contracts auto-renew every thirty days, have adjustable redundancy, and can be terminated at any time.

  • Private file encryption
  • Redundant storage nodes
  • Upload and delivery gateway

Proof of Storage

The Proof of Storage algorithim runs checks on the storage nodes to ensure your data is online and available. BROCA contracts give you control over renewal and node redundancy. Storage node runners DePIN your data in exchange for BROCA.
